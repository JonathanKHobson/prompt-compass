What is Prompt Compass?

Prompt Compass is a local prompt-engineering companion. It routes broad requests, diagnoses weak prompts, improves existing prompts, builds new prompts from intent, compares versions, and checks whether a prompt is ready for the model or surface where it will run.

Who is Prompt Compass for?

It is for people who know what they want from AI but need help turning intent into durable instructions: writers, researchers, product people, educators, analysts, developers, and technically curious users.

Does it send my prompts to a cloud service?

The MCP package is local-first. The bundled knowledge base runs locally. Personal overlay records are stored in user app data unless you configure a different path.

What is Agent Mode?

Agent Mode is an explicit advanced option that creates an approval-gated committee plan. It does not silently spawn agents, and it does not bypass host permissions.

Install Guide

  1. Download the package that matches your host.
  2. Install or upload it through that host's extension/plugin/skill flow.
  3. Restart or open a fresh task if your host does not hot-load new MCPs.
  4. Smoke test with: Use Prompt Compass and call get_started.

Safety Note

Prompt Compass should be added, not used to overwrite unrelated MCP configuration. If you install from source, preserve existing servers and use absolute paths.
